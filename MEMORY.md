# Genesis AI Memory

## Назначение

Этот документ хранит долгосрочную память проекта Genesis AI.

Его задача — сохранять знания, решения и договорённости между всеми участниками проекта, чтобы работа не зависела от памяти человека или конкретной модели ИИ.

---

# Основные сведения

Название проекта: Genesis AI

Статус:
Фаза 1 — Формирование системы.

Основная цель:
Создать ИИ-команду, способную совместно проектировать, разрабатывать и сопровождать программные продукты.

---

# Участники

CEO
Человек.
Определяет направление проекта и принимает окончательные решения.

CTO
ChatGPT.
Отвечает за архитектуру, анализ, проектирование и контроль качества решений.

Developer
Cursor (или другой инструмент разработки).
Отвечает за реализацию программного кода.

Repository
GitHub.
Хранит всю историю проекта и является единственным источником истины.

---

# Правила памяти

Сохраняются:

- архитектурные решения;
- принятые правила;
- причины важных изменений;
- выводы после завершённых задач;
- ошибки, которые нельзя повторять.

Не сохраняются:

- временные идеи;
- неподтверждённые гипотезы;
- случайные обсуждения.

---

# Принцип развития

Каждая завершённая задача должна сделать Genesis AI умнее.

Любое решение должно оставлять после себя знания.

Если информация может пригодиться в будущем — она должна попасть в память проекта.
