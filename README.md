# Genesis AI

Genesis AI — это проект по созданию ИИ-команды, в которой человек задаёт направление, а специализированные ИИ-инструменты помогают превращать идеи в работающие решения.

## Миссия

Создать практическую систему совместной работы человека и искусственного интеллекта, способную последовательно развивать проекты, сохранять знания и улучшать собственные процессы.

## Как устроена работа

- CEO — человек. Определяет цели, принимает финальные решения и несёт ответственность за направление.
- ChatGPT — стратег, архитектор и аудитор. Помогает формулировать решения, проектировать систему и проверять качество.
- Cursor — технический исполнитель. Используется для работы с кодом и файлами проекта.
- GitHub — единый источник истины для документов, истории изменений и будущего кода.

## Основные принципы

1. Сначала цель, потом инструмент.
2. Сначала план, потом изменение.
3. Решения должны быть понятными и проверяемыми.
4. Все важные знания сохраняются в репозитории.
5. ИИ помогает принимать решения, но не заменяет ответственность человека.
6. Система должна становиться полезнее после каждого рабочего цикла.

## Подход к развитию

Genesis AI развивается постепенно:

Идея → анализ → решение → реализация → проверка → фиксация знаний → следующий цикл.

Проект находится на ранней стадии формирования.
